# 🌐 Personal Portfolio Website

## 📌 Project Overview
This is a personal portfolio website built using HTML and CSS.  
It showcases my skills, projects, and contact information in a clean and responsive layout.

The portfolio includes:
- 🧑‍💻 About Me section
- 🛠️ Skills section
- 📂 Projects section
- 📞 Contact section

The main goal of this project is to present my profile and work in a professional way for recruiters and opportunities.

---

## ⚙️ Setup Instructions
1. Download or clone the repository:
   git clone https://github.com/your-username/your-repo-name.git

2. Navigate to the project folder:
   cd your-repo-name

3. Make sure all files are in the same directory:
   - portfolio.html
   - Any images used

---

## ▶️ How to Run the Project
1. Open the project folder.
2. Double-click on `portfolio.html`  
   OR  
3. Right-click the file and open with any web browser (Chrome, Edge, Firefox).

The website will open in your browser.
